LoadNIST

Load a signal with SPHERE header into MATLAB. The signal can be PCM and ULAW encoded, if compressed by SHORTEN, it will be automatically de-compressed. The byte order can also be properly swapped depending on the current machine.

Usage: Sig = LoadNIST('FName');

Example:

Sig = LoadNIST('sa1.wav');

This command loads a signal from the TIMIT database.

MFCC

Calculate Mel-scaled FFT based cepstrum (MFCC) for a given signal. MFCC parameters are given by a structure as below. The result is stored in a structure with the field name "Mat".

Analysis parameters:

(1) Par.Order : Number of MFCC coefficients.
(2) Par.Delta : 1-- MFCC and Delta MFCC, 0-- only MFCC.
(3) Par.WinSz : analysis window size.
(4) Par.StpSz : window shift step.
(5) Par.SRate : sampling rate of signal.
(6) Par.RmvSilence : remove silence part, 1-- Yes, 0 -- No.

Usage: Data = MFCC(Sig, Par);

Example:

F_Par.Order = 2;    % only two coefficients
F_Par.Delta = 0;    % No dynamic MFCC
F_Par.WinSz = 256;
F_Par.StpSz = 128;
F_Par.SRate = 16000;
F_Par.RmvSilence = 1; % remove silence part

M(1) = MFCC(Sig, F_Par);

Each row of M(1).Mat is a feature vector.      

Gen_VQ

Generate a VQ codebook by the K-means algorithm.
Training parameters are given by a structure defined as below:

    (1) Par.Size : codebook size
    (2) Par.MaxIter : maximum iterations
    (3) Par.RandSeed : random seed;


Usage: Model = Gen_VQ(Data, Par);

Example:

suppose Data(1).Mat, Data(2).Mat, ...., contain training data,

MyPar.Size = 8;
MyPar.MaxIter = 20;
MyPar.RandSeed = 1973;

VQ = Gen_Vq(Data, MyPar);

Each row of VQ.Mat is a code vector.      

 

Gen_GMM

Generate a Gaussian Mixture Model (GMM) by the EM algorithm.

Training parameters are defined as below:

    (1) Par.Size : number of Gaussian functions
    (2) Par.MaxIter : maximum iterations
    (3) Par.RandSeed : random seed;


Usage: Model = Gen_GMM(Data, Par);

Example:

suppose Data(1).Mat, Data(2).Mat, ...., contain training data,

MyPar.Size = 8;
MyPar.MaxIter = 20;
MyPar.RandSeed = 1973;

GMM = Gen_GMM(Data, MyPar);

GMM.MeanMat contains mean vectors,
GMM.VariMat contains variance vectors,
GMM.WgtMat contains mixture weigth.

Gen_CHMM

Generate continuous density hidden Markov model (HMM).

Training parameters are defined as below:

    (1) Par.Size : number of Gaussian functions
    (2) Par.MaxIter : maximum iterations
    (3) Par.RandSeed : random seed
    (4) Par.NState : number of states
    (5) Par.Conf : model structure,
     e.g. three states left-to-right [1, 1, 0; 0, 1, 1; 0, 0, 1];


Usage: Model = Gen_CHMM(Data, Par);

Example:

suppose Data(1).Mat, Data(2).Mat, ...., contain training data,

MyPar.Size = 8;
MyPar.MaxIter = 20;
MyPar.RandSeed = 1973;
MyPar.NState = 3;
MyPar.Conf = [1, 1, 1; 1, 1, 1; 1, 1, 1] % allow any transion

HMM = Gen_CHMM(Data, MyPar);

HMM.Tran : transition matrix         
HMM.MeanMat : mixture mean vectors         
HMM.VariMat : mixture variance vectors        
HMM.WgtVec : mixture weight
