Mixture model

The EM estimation in the case of mixture model is achieved through 

for i = 1:n_iter
  [gamma, logl(i)] = mix_post(X, w, mu, Sigma);
  [mu, Sigma, w] = mix_par(X, gamma, COV_TYPE);
end;
or, more simply 
[w, mu, Sigma, logl] = mix(X, w, mu, Sigma, n_iter);
