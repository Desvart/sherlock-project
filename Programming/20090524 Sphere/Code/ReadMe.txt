
Database
---------

This folder contains the various databases used in this development. It does not contain the original database, but only copies. Original versions are located at project root.


ExternalCode
-------------

This folder contains all original codes developed by external sources that have been integrated into the main code. Most of these codes have been heavily modified before being integrated. This folder is a kind of backup of the original codes.


Release
--------

This folder contains the various operational versions of project code.


Source
-------

This folder contains the project current developments and the history of past versions.


Test
------

This folder contains all tests and demonstrations codes that were conducted during development of the main code.
